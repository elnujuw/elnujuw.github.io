报错

解决方法

  1. 重新运行安装程序到图一报错的地方。

  2. 下载文件http://eddiejackson.net/apps/Fix_Error_SQL2005.zip

  3. 复制对应版本的文件到安装目录Program Files (x86) -> Microsoft SQL Server > MSSQL.2 > MSSQL > Binn替换原文件。

  4. 返回图一点击Retry。

安装完成。
